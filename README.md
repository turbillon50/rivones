# Rivones

Premium car rental platform for Mexico — Turo-style marketplace.

Built with React + Vite (frontend), Express 5 (API), PostgreSQL + Drizzle ORM, Clerk auth, Stripe payments.

## Brand
- **Domain**: rentamerapido.autos
- **Colors**: Deep navy (#0f1629), Cyan (#00d4ff)
- **Language**: Spanish (Mexico market)

## Stack
- pnpm monorepo
- TypeScript
- React + Vite + Tailwind CSS + shadcn/ui
- Express 5 + Drizzle ORM + PostgreSQL
- Clerk authentication
- Stripe payments
- Google Maps
- Framer Motion animations
- PWA support
