Build a premium mobile-first web app demo for a car marketplace platform similar to an Airbnb-style experience, but focused on used cars available in different locations.

PROJECT GOAL:
Create a visually stunning, high-conversion demo app that allows users to browse, explore, and discover used cars through a map-based interface and premium card-based UI. The app must feel like a real startup product ready to scale.

CORE IDEA:
This is a dynamic car marketplace where each car listing behaves like a “property” in Airbnb:
- geolocated
- visual
- interactive
- easy to explore
- designed to convert

IMPORTANT:
This is a DEMO first, but must be architected to support:
- real listings
- dynamic uploads
- map integrations (Google Maps)
- user interaction
- notifications
- future payments and booking/reservation flows

Use modular structure:
1. customer frontend
2. admin / seller dashboard
3. data + services layer
4. integrations layer (maps, notifications, media)

TECH STACK:
- Next.js or React
- Tailwind CSS
- Mobile-first
- PWA-ready
- Google Maps integration (mocked or placeholder)
- Mock API for cars and users
- Service files prepared for future backend
- Smooth animations
- Fast performance
